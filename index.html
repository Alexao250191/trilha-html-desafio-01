O QUE É LÓGICA DE PROGRAMÇÃO

Voltar ao topo

A lógica de programação é a forma de pensar passo a passo para resolver um problema usando instruções que o computador consegue entender. É como ensinar alguém a fazer algo sem pular etapas, de maneira clara, organizada e lógica.

Exemplo Simples:

Imagine que você quer que um robô execute a função "fazer café" para você. Para isso, você precisa dar instruções para ele como:

  1. Pegue o coador e coloque o filtro;
  2. Coloque o pó de café dentro do filtro;
  3. Aqueça a água;
  4. Despeje a água no filtro onde estava o pó de café;
  5. Após acabar de filtrar, sirva na xícara.

Caso você esqueça, inverta ou não siga essa sequência, o robô irá se perder no processo de fazer café. Com o código de programação é a mesma coisa: todo o passo a passo de como tem que ser executado deve ser minunciosamente detalhado.

Elementos Importantes da Lógica de Programação

Voltar ao topo

Como já foi descrito anteriormente, a lógica da programação versa sobre a forma de pensar passo a passo para resolver um determinado problema, sem pular etapas e de maneira clara, organizada e lógica. A cerca desse assunto, vamos detalhar alguns elementos super importantes a respeito da lógica de programação. São eles:

  1. Sequência

    2. É a ordem correta das instruções, o passo a passo propriamente dito.

    Ex.: Primeiro abrir uma caixa depois retirar o conteúdo

  2. Decisão (Condicionais)

    3. Trata sobre a decisão da ação a ser realizada se algo for verdadeiro. "se sim" faça tal coisa, "se não" faça outra coisa.(if, else)

    Ex.: Abra a caixa e, se tiver algo dentro, tire-o. Se não tiver nada dentro, feche-a

    O mesmo pode acontecer com senhas. Se a senha estiver correta você pode acessar, mas se estiver errada então você não poderá acessar.

  3. Repetição

    4. Repete a mesma ação ou sequência de ações por várias vezes.

    Ex.: Enquanto a chaleira conter água, continue fervendo

  4. Variáveis

    5. As variáveis são "caixinhas" onde se é guardado informações temporárias e que mudam (variam) de um usuário para outro.

    Ex.: Em um fomrulário de cadastro contém várias informações variáveis.

    Nome = Bruce Wayne, Profissão: vigilante noturno

  5. Funções

    6. São conjuntos de ações que podem ser utilizadas várias vezes. Como a função "fazer café" descrita lá no início do texto.

    Ex.: Uma função chamada "FazerCafe" pode ser chamada toda vez que você quiser café

Por que a lógica é tão importante?

A lógica é um componente muito importante para a resolução de todo e qualquer problema do nosso cotidiano, seja na vida real ou na programação. Um código lógico é muito mais fácil de enterder e de manter, a lógica te auxilia a entender o problema antes de escrever o código para resolvê-lo, além de evitar travamentos ou bugs e, a lógica é algo universal e você poderá utilizar no seu cotidiano e em outras linguagens de programação também.

O que é Pensamento Computacional

Voltar ao topo

O pensamento computacional é uma habilidade essencial para quem deseja aprender programação, mas vai muito além do mundo dos códigos. Ele é a capacidade de pensar de forma estruturada, lógica e eficiente para resolver problemas — do dia a dia ou dentro do computador.

Pensamento computacional é como organizar a mente para resolver problemas como um computador faria: por partes, com clareza, foco no essencial e com passos bem definidos. Essa forma de pensar é usada na programação, mas também serve para montar uma receita, planejar uma viagem ou organizar uma tarefa em etapas.

Os 4 Pilares do Pensamento Computacional

Voltar ao topo

O pensamento computacional compreende de 4 pilares para sua eficácia na organização dos dados e resulução de um dado problema. Esses pilares são:

  1. Decomposição

    Dividir um problema grande em partes menores é mais fácil de resolver

      Ex.: Criar um site pode ser dividido em montar o menu, criar o conteúdo, aplicar os estilos, configurar as interações, etc.
  2. Reconhecimento de Padrões

    Observar semelhanças e repetições para economizar tempo e esforço.

      Ex: Vários sites têm cabeçalho e rodapé parecidos. Use o mesmo modelo.
  3. Abstração

    Focar só no que importa, deixando de lado os detalhes desnecessários.

      Ex: Ao planejar um sistema de login, o importante é o nome e senha, não a cor do botão.
  4. Algoritmo

    Criar uma sequência clara de passos para alcançar um objetivo.

      Ex: Para acessar um sistema: digitar login, depois digitar senha, depois clicar em entrar, depois acessar a conta.

Por que ele é importante para a programação?

Porque programar não é só escrever comandos. É pensar antes de escrever, planejar o que o código deve fazer, e prever situações. O pensamento computacional ajuda a:

  • Organizar melhor as ideias;
  • Planejar soluções antes de codar (sintetizar o código);
  • Enetender problemas complexos e dividí-los em etapas simples;
  • Criar códigos mais claros, eficientes e com menos erros.

Mas o que é HTML?

HTML significa HyperText Markup Language, ou Linguagem de Marcação de Hipertexto. Em resumo, ela não é considerada uma linguagem de programação, dado ao fato de que ela é uma linguagem utilizada para estruturar o conteúdo de uma página de web através de algumas ferramentas chamadas de tags (e alguns atributos). Essas tags dizem ao navegador o que cada parte do conteúdo representa: Título, parágrafo, tabela, imagem, etc.

O HTML é responsável por estruturar a página através de algumas ferramentas , definindo o que é cada item e como o browser (navegador) irá ler aquele item e mostrar na sua tela. Vou dar um exemplo: visualize uma casa. O HTML seria como os blocos e colunas de uma casa, o CSS (é outra linguagem de programação) seria a decoração e pintura, e o JavaScript (também outra linguagem de programação) seria a eletricidade e automação (que dão vida à casa).

Os conceitos da lógica e do pensamento computacional são importantíssimos aqui no HTML para escrever em ordem lógica e funcional a estruturação da página. colocando em seus devidos lugares cada item.

Portanto, podemos concluir que o HTML não é uma linguagem de programação porque ele não executa ações, ele apenas estrutura todas elas definindo o conteúdo de páginas web. Porém, é a parte essencial para desenvolver códigos para web já que ele é a base de todo site. E que sem os fundamentos de lógica e pensamento computacional, seu site pode ficar difícil de ler, cheio de bugs e seu código complicado de compreender.

O que são Client e Server?

Voltar ao topo

A comunicação entre cliente e servidor (Client x Server) é um dos conceitos mais importantes da programação, especialmente na web. Entenda cada um deles:

Cliente (Client)

É quem faz o pedido e pode ser o seu navegador, um aplicativo, até uma geladeira inteligente.

Servidor (Server)

É quem recebe o pedido e envia uma resposta. Geralmente é um computador ou sistema na internet que armazena sites, dados ou arquivos, processa os pedidos e devolve informações.

Exemplo:

  1. Você liga para a pizzaria: você é o cliente;
  2. A pizzaria atende, anota o pedido e faz a pizza: ela é o servidor;
  3. A pizzaria entrega a pizza na sua casa: Essa é a resposta.
Assim funciona a web:
  • O cliente (navegador) faz uma requisição ao servidor;
  • O servidor recebe, processa e envia uma resposta conforme a requisição.
Esse é o conceito básico sobre o funcionamento da internet. Entender esse processo é essencial para criar sites, sistemas web ou aplicativos. Nesse contexto, o cliente acessa um site em seu dispositivo e essa requisição vai para o servidor que, processa o arquivo HTML e devolve para o usuário o site solicitado.
